# Quick Search on Spotify/SoundCloud Chrome Extension

<img src="../SearchOnSoundcloud/images/soundcloud-128px.pnh" width="100" /> <img src="../SearchOnSpotify/images/spotify-128px.pnh" width="100" />

This Chrome Extension allows users to highlight text on any webpage and quickly search the selected text on Spotify or SoundCloud by right-clicking and selecting the "Search with Custom URL" option from the context menu. It's designed to provide music lovers with a seamless and instant way to find and listen to their favorite artists and tracks on their preferred streaming platform without manually opening the site and typing their search query.

## Screenshot

![Quick Search Extension in Action](screenshot)

## Features

- **Easy to Use**: Simply highlight, right-click, and search.
- **Quick Access**: Instantly opens a new tab with search results on Spotify or SoundCloud.
- **Flexible**: Works on any website, making it easier to discover music while browsing.

## How It Works

1. Highlight the text you want to search for (e.g., artist name, song title).
2. Right-click to open the context menu.
3. Select "Search on Spotify" or "Search on SoundCloud".
4. A new tab will open with your search results on the selected platform.

## Installation

1. Download the ZIP file of the extension from this repository.
2. Unpack the ZIP file on your computer.
3. Open Chrome and navigate to `chrome://extensions/`.
4. Enable "Developer mode" at the top right.
5. Click "Load unpacked" and select the unpacked extension folder.
6. The extension is now ready to use!
